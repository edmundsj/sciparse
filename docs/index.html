sciparse API¶

unit_parsing.column_from_unit(data, unit)¶

Extracts the pandas column corresponding to a particular unit quantities. If there are multiple columns, it gives back the first one. Assumes columns names are of the form “Quantity Name (unit)”, i.e. Voltage (mV) or Photocurrent (nA).

Parameters

  • data – Pandas dataframe to extract column from

  • unit – Pint unit of the desired column. Data will be returned in terms of this unit

unit_parsing.sampling_period(data)¶

Extracts the sample period from a pandas DataFrame assuming that data is evenly spaced in time. Uses the difference in time between the first and second data points to determine the sampling period and sampling frequency.

Parameters

data – Pandas array with Time (including units, i.e. Time (ms))

Returns sampling_period

Sampling period as a pint quantity

parsers.parse_default(filename, data=None, metadata=None, read_write='r')¶

Default parser for data. Assumes there is a single line of metadata at the top line in the form of a dictionary and the remainder is tabular and can be imported as a pandas DataFrame

Parameters

  • filename – Name of the file to be written

  • data – Data to write to file

  • metadata – Metadata to write to file

  • read_write – “r” or “w”. Read or write.
